# 雨滴-Raindrops

把一个数字转换成一个字符串,它的内容取决于，数字的因子.

- 如果数字有 3 作为一个因子,输出"Pling".
- 如果数字有 5 作为一个因子,输出"Plang'".
- 如果数字有 7 作为一个因子,输出"Plong".
- 如果数字没有 3, 5,或 7 作为一个因子, 直接给数字。

## 实例

- 28 的因子是 1, 2, 4, **7**,14, 28.
  - 雨滴说,这将是一个简单的"Plong".
- 30 的因子是 1, 2,**3**,**5**,6, 10, 15,30.
  - 雨滴说,这将是一个"PlingPlang".
- 34 有四个因子:1, 2, 17,34.
  - 雨滴说,这将是"34".
